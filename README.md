# vityarthiproject
This is a python program that provides healthy food recommendations based on the body mass index,age, amount of calories and number of steps walked on a particular day.It recommends all three meals of the day breakfast,lunch nad dinner.The aim is to help users make healthier food choices using simple data .The code works by using input such as height,weight,steps,calories consumed and age.It calculates the bmi using the height and weight and classifies the user as underweight ,normal weight or over weight and suggests three meal plans breakfast lunch nad dinner.
All the meal recommendatins are designed to be suitable for the calories range and easy enough to be followed by young adults as well as teenagers.
Features of teh program:
calculates the body mass index
categorises the calorie intake 
uses age and steps to improve recommendation
recommends a nutritionally balanced meal plan based on all the provided inputs
